# Sistem E Learning Sederhanan REST API

Sistem ini menggunakan framework Laravel versi 12 dan sudah di lengkapi Sanctum untuk autentikasi berbasis token.

## API Reference

Berikut beberapa list API yang ada dalam sistem ini

#### Login

```http
  POST /api/login
```

#### Register

```http
  POST /api/register
```

#### Logout

```http
  POST /api/logout
```

#### Get Data Course

```http
  GET /api/courses
```

#### Post Course

```http
  POST /api/courses
```

#### Edit Course

```http
  PUT /api/courses/{id}
```

#### Delete Course

```http
  DELETE /api/courses/{id}
```
